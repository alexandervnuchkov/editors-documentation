Insert headers and footers

Headers and footers allow to add some additional info on a slide, such as date and time, slide number, or a text.

To insert a header or footer in a presentation:

  1. switch to the Insert tab,
  2. click the Edit Header/Footer button at the top toolbar,
  3. the Header/Footer Settings window will open. Check the data you want to add into the footer. The changes are displayed in the preview window on the right.
    • check the Date and time box to insert a date or time in a selected format. The selected date will be added to the left field of the slide footer.

      Specify the necessary data format:

      • Update automatically - check this radio button if you want to automatically update the date and time according to the current date and time.

        Then select the necessary date and time Format and Language from the lists.

      • Fixed - check this radio button if you do not want to automatically update the date and time.
    • check the Slide number box to insert the current slide number. The slide number will be added in the right field of the slide footer.
    • check Text in footer box to insert any text. The text will be added in the central field of the slide footer.

  4. check the Don't show on the title slide option, if necessary,
  5. when ready, click the OK button to apply the changes.

To edit the added footer, click the Edit Header/Footer button at the top toolbar, make the necessary changes in the Header/Footer Settings window, and click OK to save the changes.
