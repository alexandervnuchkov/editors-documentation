Добавление подложки

Watermark allow to indicate your document status (for example, confidential, draft etc.).

Для добавления подложки в документ:

  1. перейдите на вкладку Макет на верхней панели инструментов,
  2. нажмите на значок Подложка на верхней панели инструментов и выберите пункт меню Настраиваемая подложка,
  3. откроется окно Параметры подложки window will appear where you can specify the watermark parameters,

  4. Выберите тип подложки, которую требуется вставить:
    • Используйте опцию Текстовая подложка и настройте доступные параметры:
      • Язык - выберите из списка один из доступных языков,
      • Текст - выберите один из доступных примеров текста на выбранном языке. Для русского языка доступны следующие тексты подложки: ДЛЯ СЛУЖЕБНОГО ПОЛЬЗОВАНИЯ, ДСП, КОПИРОВАТЬ НЕ РАЗРЕШАЕТСЯ, КОПИЯ, ЛИЧНОЕ, ОБРАЗЕЦ, ОРИГИНАЛ, СЕКРЕТНО, СОВ. СЕКРЕТНО, СОВЕРШЕННО СЕКРЕТНО, СРОЧНО, ЧЕРНОВИК.
      • Шрифт - select the font name and size from the corresponding drop-down lists. Use the icons to set the font color or apply one of the decoration styles: Bold, Italic, Underline, Strikeout,
      • Полупрозрачный - check this box if you want to apply transparency,
      • Расположение - select the По диагонали or По горизонтали option.
    • Используйте опцию Графическая подложка и настройте доступные параметры:
      • Choose the image file source using one of the buttons: Из файла или По URL - the image will be displayed in the preview window on the right,
      • Масштаб - select the necessary scale value from the available ones: Авто, 500%, 200%, 150%, 100%, 50%.
  5. Нажмите кнопку OK.

To edit the added watermark, open the Watermark Settings window as described above, change the necessary parameters and click OK.

To delete the added watermark click the Watermark icon at the Layout tab of the top toolbar and choose the Remove Watermark option from the menu. It's also possible to use the None option in the Watermark Settings window.
